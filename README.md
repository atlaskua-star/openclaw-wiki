## 🛠 Tech Stack

- **React 18** — UI framework
- **TypeScript** — type safety
- **Vite 5** — build tool
- **react-markdown** + **remark-gfm** — markdown rendering
- **Dark theme** — minimal, blue accent (#3b82f6)

## 💻 Local Development

```bash
npm install
npm run dev     # dev server
npm run build   # production build
```
